## Dificuldades encontradas ao usar o Vim ou Nano:
Saber os comandos certinhos para instalar e fazer upgrade do "VIM"
># Comandos que mais utilizei:
>pwd: Mostra o caminho da pasta atual.
>ls: Lista arquivos e pastas.
>cd: Entra em pastas.
>mkdir: Cria pastas.
>touch: Cria arquivos.
>cp: Copia arquivos ou pastas.
>mv: Move ou renomeia arquivos ou pastas.
>rm: Remove arquivos (cuidado!).
>nano / vim: Editores de texto internos.
